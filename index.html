Política de Privacidade — LDRT - Pesquisa

Última atualização: 12 de abril de 2025

O aplicativo “LDRT - Pesquisa” foi desenvolvido com o objetivo de facilitar o acesso rápido e intuitivo à lista de doenças relacionadas ao trabalho, com base na PORTARIA GM/MS Nº 1.999, de 27 de novembro de 2023 — para fins didáticos. Esta política de privacidade descreve como as informações dos usuários são tratadas.

1. Coleta de Dados

Este aplicativo não coleta, armazena, compartilha nem processa quaisquer dados pessoais dos usuários.

  • Não é necessário criar conta ou login.
  • Nenhum dado é transmitido a servidores externos.
  • Nenhuma informação pessoal, de localização, dispositivo ou uso é coletada.
  • O aplicativo funciona 100% offline.

2. Uso de Dados

Como o aplicativo não coleta dados, também não utiliza dados para nenhum fim — seja comercial, analítico ou técnico.

3. Permissões do Dispositivo

O aplicativo pode solicitar permissões mínimas apenas para garantir seu funcionamento adequado, mas nenhuma informação é coletada a partir dessas permissões.

4. Publicidade e Compras

O aplicativo é totalmente gratuito e não contém anúncios ou qualquer forma de compra interna (in-app purchases).

5. Conteúdo e Avisos Legais

Este aplicativo é um projeto independente, sem vínculos oficiais com qualquer órgão público ou privado. Todo o conteúdo é baseado em interpretação da PORTARIA GM/MS Nº 1.999/2023 e se destina a consulta preliminar. Para decisões profissionais, recomenda-se a consulta direta à portaria oficial.

6. Alterações nesta Política

Podemos atualizar esta política de tempos em tempos. Recomenda-se que o usuário consulte esta página periodicamente para estar ciente de quaisquer mudanças.
