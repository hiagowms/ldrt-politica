Política de Privacidade — LDRT - Pesquisa

Última atualização: 13 de abril de 2025

O aplicativo “LDRT - Pesquisa” foi desenvolvido com a finalidade de apresentar de forma dinâmica uma lista de informações que visa inspirar o interesse no tema da saúde do trabalhador. Todo o conteúdo disponível no aplicativo é fruto do entendimento pessoal do autor e destina-se exclusivamente a fins de inspiração à estudar o tema, não devendo ser utilizado como fonte oficial ou para tomar qualquer decisão profissional.

1. Coleta de Dados

Este aplicativo não coleta, armazena, compartilha ou processa quaisquer dados pessoais dos usuários. Em específico:

  • Não é necessário criar conta ou efetuar login.
  • Nenhum dado é transmitido para servidores externos.
  • Nenhuma informação pessoal, de localização, dispositivo ou uso é coletada.
  • O aplicativo funciona 100% offline.

2. Uso de Dados

Como o aplicativo não coleta dados, não há utilização de informações para quaisquer fins — sejam eles comerciais, analíticos ou técnicos.

3. Permissões do Dispositivo

O aplicativo não solicita nenhuma permissão ao usuário, pois se trata apenas de uma lista de informações acessível integralmente offline.

4. Publicidade e Compras Internas

O aplicativo é totalmente gratuito, isento de anúncios e não possui funcionalidades de compras internas (in-app purchases).

5. Conteúdo, Disclaimer e Avisos Legais

Todo o conteúdo deste aplicativo é fruto do entendimento pessoal do autor e foi desenvolvido com o objetivo único de inspirar o interesse no tema da saúde do trabalhador.

AVISO IMPORTANTE: O conteúdo apresentado neste aplicativo não se destina a ser utilizado como fonte oficial, tampouco serve para substituição de orientações profissionais ou consulta a fontes primárias. Recomenda-se que todas as informações sejam verificadas junto a fontes especializadas e oficiais para quaisquer finalidades práticas.

6. Alterações nesta Política

Esta política de privacidade poderá ser atualizada periodicamente. Recomendamos que os usuários verifiquem esta página regularmente para se manterem informados sobre possíveis alterações.
